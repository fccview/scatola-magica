# Supported File Types

## Viewable Files

The following file types can be opened and viewed in the app:

### Text Files (View & Edit)

These files can be viewed with syntax highlighting and edited directly in the app:

- `.txt` - Plain text files
- `.md`, `.markdown` - Markdown files
- `.html` - HTML files
- `.css` - CSS stylesheets
- `.js`, `.jsx` - JavaScript files
- `.ts`, `.tsx` - TypeScript files
- `.json` - JSON data files
- `.xml` - XML files
- `.yaml`, `.yml` - YAML configuration files
- `.sh`, `.bash` - Shell scripts
- `.py` - Python files
- `.java` - Java files
- `.c`, `.cpp`, `.h`, `.hpp` - C/C++ files
- `.go` - Go files
- `.rs` - Rust files
- `.php` - PHP files
- `.rb` - Ruby files
- `.sql` - SQL files
- `.log` - Log files
- `.config`, `.conf`, `.ini`, `.env` - Configuration files

### Documents (View Only)

- `.pdf` - PDF documents
- `.csv` - CSV spreadsheets (displayed as tables)

### Images (View Only)

- `.jpg`, `.jpeg` - JPEG images
- `.png` - PNG images
- `.gif` - GIF images
- `.svg` - SVG vector graphics
- `.webp` - WebP images
- `.bmp` - Bitmap images
- `.ico` - Icon files

### Videos (View Only)

- `.mp4` - MP4 videos
- `.webm` - WebM videos
- `.ogg` - Ogg videos
- `.mov` - QuickTime videos
- `.avi` - AVI videos
- `.mkv` - Matroska videos

## Opening Files

You can open supported files by:

- **Double-clicking** on a file
- **Right-clicking** and selecting "Open" from the context menu
